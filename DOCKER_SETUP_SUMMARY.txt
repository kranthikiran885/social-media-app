================================================================================
                         QUICK START GUIDE
================================================================================

STEP 1: Clone & Setup
  $ git clone <repo-url>
  $ cd social-media-app
  $ cp .env.production .env.local
  $ nano .env.local  # Edit with your configuration

STEP 2: Configure Secrets
  Edit .env.local and set:
  - JWT_SECRET (random 32+ chars)
  - MONGO_PASSWORD (strong password)
  - POSTGRES_PASSWORD (strong password)
  - AWS credentials (if using S3)
  - Stripe keys (if using payments)
  - OpenAI key (if using AI features)

STEP 3: Deploy
  $ chmod +x deploy.sh
  $ ./deploy.sh

STEP 4: Verify
  $ docker-compose ps
  $ curl http://localhost/health
  $ curl http://localhost:8000/health

STEP 5: Access Services
  Frontend: http://localhost
  Grafana: http://localhost:3000 (admin/password)
  Prometheus: http://localhost:9090
  RabbitMQ: http://localhost:15672

================================================================================
                    COMMON COMMANDS
================================================================================

Deployment:
  ./deploy.sh deploy         # Full deployment
  ./deploy.sh stop           # Stop all services
  ./deploy.sh restart        # Restart services
  ./deploy.sh status         # Show status
  ./deploy.sh logs           # View logs

Docker Compose:
  docker-compose -f docker-compose.production.yml up -d
  docker-compose -f docker-compose.production.yml down
  docker-compose -f docker-compose.production.yml logs -f
  docker-compose -f docker-compose.production.yml ps
  docker-compose -f docker-compose.production.yml restart auth-service

Database Access:
  # MongoDB
  docker exec -it smart_social_mongodb mongosh

  # PostgreSQL
  docker exec -it smart_social_postgresql psql -U postgres

  # Redis
  docker exec -it smart_social_redis redis-cli

Scaling Services:
  docker-compose -f docker-compose.production.yml up -d --scale chat-service=3

View Logs:
  docker-compose -f docker-compose.production.yml logs -f auth-service
  docker-compose -f docker-compose.production.yml logs --tail=100

                      TROUBLESHOOTING
================================================================================

Service Won't Start:
  $ docker-compose logs service-name
  $ docker-compose ps
  $ docker-compose build --no-cache service-name

Connection Issues:
  $ docker network ls
  $ docker network inspect smart_social_network
  $ docker exec auth-service curl http://mongodb:27017

Memory Issues:
  $ docker stats
  # Edit docker-compose and set memory limits

Log Problems:
  $ docker-compose logs -f service-name
  $ docker-compose logs --tail=100
  $ docker-compose logs --since 2024-01-15T10:00:00
